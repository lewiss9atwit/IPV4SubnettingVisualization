# IPV4 Subnetting Visualization Project


A javaFX program that calculates and visualizes IPV4Subnetting

## Run Instructions

1. Install [JavaFX SDK](https://openjfx.io).
2. In Eclipse, set the VM arguments:
--module-path "path/to/javafx-sdk/lib" --add-modules javafx.controls,javafx.fxml
